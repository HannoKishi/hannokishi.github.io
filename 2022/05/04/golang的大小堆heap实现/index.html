在leetcode刷题时候，会用到大小堆的数据结构。在go语言中，大小堆的实现是有现成的接口的。

container/heap包

在go语言中，container/heap包中存在一个接口。

要实现该接口需要实现具体的方法。

  • sort.Interface接口（sort包中）

因此要实现一个指定数据结构的大小堆需要实现一下的方法：

  • Push(x any) : 增加一个元素
  • Pop() any : 返回一个元素
  • Len() int : 返回长度
  • Less(i, j int) bool : 元素大小比较方法
  • Swap(i, j int) : 交换元素方法
